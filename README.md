# Employee Management Portal - Basic CRUD Application
It aims to showcase the development of an Employee Management Portal. This portal allows for the management of employee data through a CRUD interface. Users can add new employees, view a list of employees, update employee details, and delete employees from the system.

# Features

- Add Employee: Add new employee records to the system.
- View Employees: Display a list of all employees with their details.
- Update Employee: Modify existing employee information.
- Delete Employee: Remove an employee from the system.

## Installation 

The project can be installed from the repository by following the given steps :

1. Cloning the repository :
```bash
git clone https://github.com/sarthakkurothe/vahan-assignment.git
```
2. Change it to project directory :
```bash
cd vahan-assignment-main
```

- To run the backend server :

1. Change it to backend directory :
```bash
cd backend
```
2. Install the required dependencies :
```bash
npm install
```
3. Run the backend server :
```bash
npm run dev
```

- To run the frontend server :

1. Change it to frontend directory :
```bash
cd frontend
```
2. Install the required dependencies :
```bash
npm install
```
3. Run the frontend server :
```bash
npm start
```
